Al registrar una serie o película vas a poder buscarla y traer automáticamente género, tipo, portada y capítulos totales. Series/anime usan Jikan (API de MyAnimeList, no necesita clave). Para buscar en TMDB (series live action y películas) necesitás una clave gratuita propia.
Conseguila gratis en themoviedb.org/settings/api (creá una cuenta → Configuración → API → "Solicitar una clave de API" → tipo "Developer"). Se guarda solo en este dispositivo/tu backup, nunca se envía a ningún otro lado.
